Creating a network diagram

Prerequistes:

It is recommended that you read the help files installed with Visual Notes to help you grasp a full understanding of the availalble tools. To access the help files, startup Visual Notes from the start menu or the /windows/startup/programs/ directory. Select help from the start menu and the Visual Notes help file should open up. Visual Notes help is also accessible from your Pocket PC's general help page. Go to the today screen. Select help from the start menu. Select visual notes from the help page.

Lets Get Started:

Step A: Creating the bitmaps

1. Select the create shape tool from the general tool bar.

2. Select  network tool set 1 from the drop-down tool-set selection box.

3. Select the client tool from the network diagram tool bar.

4. Tap and hold the stylus in the location of the top-left corner of the client bitmap seen below to create "client1".

5. Drag the pen across the stylus down and to the right to create the new bitmap's bounding rectangle.

6. Lift the pen off the stylus and watch the new shape appear. A bitmap's bounding rectangle is given a static size no matter how large you indicated its bounding rectangle to be.

7. Tap and hold the stylus in the location of the top-left corner of the second client bitmap seen below to create "client2".

8. Perform steps 5 and 6.

9. Tap and hold the stylus in the location of the top-left corner of the third client bitmap seen below to create "client3".

10. Perform steps 5 and 6.

11. Select the cloud tool from the network diagram tool bar.

12. Tap and hold the stylus in the location of the top-left corner (not visible) of the cloud seen below to create "cloud".

13. Pefrom steps 5 and 6. A cloud is not a bitmap, by default, you can give it an arbitrary size.

14. Select nothing else but the show-properties tool. The cloud's dialog box will appear.

15. Tap the button labeled Bkgnd.

16. Select the white color (the bottom-right most button). A second dialog box will open.

17. Click OK to save changes and exit second dialog box.

18. Click the button labeled More>>.

19. In the caption field, enter "network".

20a. Click OK save changes and exit.

20b. If neccessary, select the resize tool . Select the previous shape. Drag handles to resize shape so that the entire caption is displayed.

21. Select the server tool from the network diagram tool bar.

22. Tap and hold the stylus in the location of the top-left corner of the server bitmap seen below to create "server".

23. Perform steps 5 and 6.

24. Select nothing else but the show-properties tool. The bitmap's dialog box appears.

25. Select "Mainframe Server" in the listbox. The preview window will display a picture of the corresponding bitmap. To actually make a bitmap the current bitmap, you must double tap the listbox item. Double tap "Mainframe Server".

26. Click OK
