Creating a Flow Chart

 

Prerequistes:

It is recommended that you read the help files installed with Visual Notes to help you grasp a full understanding of the availalble tools. To access the help files, startup Visual Notes from the start menu or the /windows/startup/programs/ directory. Select help from the start menu and the Visual Notes help file should open up. Visual Notes help is also accessible from your Pocket PC's general help page. Go to the today screen. Select help from the start menu. Select visual notes from the help page.

 

Let's Get Started:

Startup visual notes from the start menu or from the /windows/startmenu/programs/ directory. Select new from the application menu a let's begin.

Step A: Creating the Shapes

1. Select the create shape tool from the general tool bar.

2. Select the flow chart tool-set from the drop-down tool-set selection box.

3. Select the rectangle tool from the flow chart tool bar.

4. Tap and hold the stylus in the location of the top-left corner of the rectangle seen below to create "rectangle1".

        

5. Drag the pen across the stylus down and to the right to create the new shape's bounding rectangle.

6. Lift the pen off the stylus and watch the new shape appear.

7. Tap and hold the stylus in the location of the top-left corner of the rectangle seen below to create "rectangle2".

        

8. Perform steps 5 and 6.

9. Select the diamond tool from the flow chart tool bar.

10. Tap and hold the stylus in the location of the top-left corner (not visible) of the diamond seen below to create "diamond".

       

11. Perform steps 5 and 6.

12. Select the shear tool from the flow chart tool bar.

13. Tap and hold the stylus in the location of the top-left corner(not visible) of the shear seen below to create "shear".

      

14. Perform steps 5 and 6.

 

Step B: Changing the Shape's Properties

1. Select the pick tool from the general tool bar.

2. Select the first symbol you created ( rectangle1 ).

3. Select the show-properties tool from the general tool bar. The symbol's dialog box should appear.

4. In the dialog box, select More>>. A second dialog box should appear.

5. In the caption field, enter "recieve order".

6. Click OK to exit and save changes in the second dialog box.

7a. Click OK to exit and save changes in the first dialog box ( saves changes for both second and first dialog boxes ). You should see the new caption centered within the text.

7b. If neccessary, select the resize tool . Select the previous shape. Drag handles to resize shape so that the entire caption is displayed. Select the pick tool again.

      

8. Select the second shape you created ( rectangle2 ).

9. Perform steps 4 through 7, except, enter "process order" in the caption field.

10. Select the third shape you created ( diamond ).

11. Perform steps 4 through 7, except, enter "cash" in the caption field.

12. Select the fourth and final symbol you created ( shear ).

13. Perform steps 4 through 7, except, enter "verify credit" in the caption field.

      

Setp C: Creating Edges

1. Select the create-edge tool from the general tool bar.

2. Select edge tool set from the drop-down tool-set selection box.

3. Select the direct edge tool from the edge tool bar.

4. Go toTools | Settings and turn on Direct Edge Axis Aligned. This will force the direct edge to always be either verticle or horizontal.

5. Click OK to exit and save changes. 

6. Select the bottom-most edge of  "rectangle1". A blue, dashed rectangle will enclose the shape to indicate selection.

7. Select the top corner of "diamond" to finsish the connection of  "straight edge1".

     

8. Select the right corner of "diamond".

9. Select the center of the left-most portion of  "shear" to complete "straight-edge3".

10. Select the bottom corner of the "diamond".

11. Select the top most portion of  "rectangle2" to complete "straight-edge3"

12. Select the corner edge tool from the edge tool bar.

13. Select the right most portion of  "rectangle2".

14. Select the bottom most portion of  "shear" to complete "corner-edge".

15. The direction of flow from "rectangle2" to "shear" is pointing the wrong way. To change this select the pick tool.

16. Select "corner-edge". The entire edge is highlighted to indicate selection.

17. Select the show-properties tool . The edge's dialog box will appear.

18. Change the master's joint symbol to arrow and the slave's joint symbol to none.

19. Click OK to save the changes.

20. Deselect the pick tool by tapping it.

       

 

Step D : Adding Labels.

1. Select the create label tool from the general tool bar.

2. Tap the area just above "rectangle2" and left of "straight-edge3". "Label" should appear in the area you tapped.

3. Select the pick tool and chose "Label ".

4. Select the show-properties tool from the general tool bar and the label's dialog appears. enter "Yes " in the label text field.

5. Click OK to save changes and exit.

6. Again, select the label tool , then tap the area the left of "shear" and just above "straight-edge2 ".

7. Perform steps 3 through 5, except, enter "No " in the label text field.

8. Select the create-hierarchy tool   from the general tool bar.

9. Select the "Yes " label, this will be the child. A red, dashed rectangle will surround the label to indicate selection.

10. Select "rectangle2 " to complete the hierachial relationship. This will be the parent.

11. Select the "No " label to be a child.

12. Select "shear " to be its parent.

13. Select the resize tool and, from its center, drag "rectangle2" to the left. This "Yes " label will have also moved since it is its child.

14. Select the undo tool to undo your changes.

       

 

Step E: Creating a title.

Currently, there isn't room for the title so we would like to move the entire diagram we created towards the bottom of the screen.

1. Select the pick tool .

2. Select "rectangle1", "rectangle2", "shear", and "diamond".

3. Tap and hold any of the selected symbols until a rectangle appears that surrounds all selected symbols.

4. Drag this rectangle towards the bottom of the screen until "rectangle1" is about 1/4 down the screen.

5. Select the create-label tool from the general tool bar.

6. Tap the canvas area just below the create-label tool.

    

7. Select nothing else but the show-properties tool from the general tool bar. This is an example of a quick shortcut for when you  become accustomed to Visual Notes. Just after you create a new symbol you can immediately change it's properties without having to select it first.

8. Enter "Acme Office" in the label text field.

9.Select a font size of "13".

10.Close the input panel (keyboard) and select underline.

11.Tap the black square whose label says color. Pick any color you want the label to be then click OK to save and exit.

12.Click OK to save changes and exit.

13. Select resize tool .

14. Drag label from it's center to reposition on the canvas. Drag the bottom-right handle towards the right until there is enough room for "Acme Office " to fit on one line.

15. Tap the resize-view   tool to gain a larger view. Note, this is NOT the same as resizing the canvas.

Step F: We're Done!

1. Select Tools | Save A s from the application menu. Save the file as "My FlowChart" and click OK

2. Click OK to exit the application.

Note: Your work will always be saved when you close a new file which has been updated. If you don't chose a file name, your work will be saved as "MyDiagramx" where x is any arbitrary number.
